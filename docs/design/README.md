ce示例所需物品配置为：
短剑蓝图：蓝色图纸，中间一个像素短剑，名称为 蓝图: 短剑
短剑: 名称为短剑
这里用于编写源质锻造系统的详细设定。

建议先从这些部分写：

- 装备类型：短剑、长剑、大剑等
- 装备等级：1-5 级的基础属性和解锁条件
- 蓝图：每种蓝图对应什么装备类型
- 基础材料：必需材料和数量
- 额外素材：影响词条数量、品质、概率、数值范围
- 词条池：普通、稀有、特殊词条
- CraftEngine 物品 ID：蓝图、材料、装备模板
- ChunkWorld 等级：装备写入的 `chunkworld:level` 规则
- pixelshop 物品、装备的价格设定，其实也是写入pdc

在chunkworld里和pixelshop里调用此插件的物品需要兼容一下格式才行

一个装备，指令打开一个界面（后续绑定到一个ce家具上）
短剑蓝图（名称为 蓝图: 短剑，lore添加蓝图等级，随机的1-3级 或固定的2级这种，然后所需锻造材料，例如铁锭x2，1级源质，木棍x1）
然后对应的材料
然后是特殊材料（此时用剧毒药剂）
就能合成出成品了
